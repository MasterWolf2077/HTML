Any fool can write code that a computer can understand. Good programmers write code that humans can understand. -Martin Fowler


A quote from Robert C. Martin's book on clean code principles

In the journey to becoming a better coder, one of the best practices is keeping your code clean and readable. imagine you are writing your code not just for computers but for humans as well. This will not only improve your coding skills but also make maintenance easier.

I recently read Clean Code: A Handbook of Agile Software Craftsmanship- Robert C. Martin

Here's an example of clean code:

function calculateArea(radius) {
        return Math.PI * radius * radius;    
    }

Press Ctrl + S to save your work regularly.

Sample output: Area claculated: 78.54 (assuming radius is 5).
